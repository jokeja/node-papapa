听得此言，才有人高喊停手，稀疏的箭雨方才停了下来。一匹战马迎了上来，马上端坐着一位披挂亮银甲的年轻将军，眉清目秀，甚是英俊，手执着一杆梅花亮银枪，膝下一匹白龙马，端的威风凛凛。非是别人，正是于一龙。
看到武天骄来了，于一龙眉头皱了一皱，淡然道：“金刀驸马，怎么就你一个人来？高玉寒大姐呢？她们不是和你在一起吗？”
说话间，脸上冷冰冰的，面无表情，不苟言笑，仿佛武天骄欠了他二五百的金币没还似的。
“她们都被驼牛群冲散了，我也不知道她们现在在哪！”
武天骄信口胡扯，探头向小山上仰望，道：“公主姐姐在哪里？她有没有事？”
听到这话，于一龙的脸色变了，张了张嘴，没有说出话来。武天骄见状心头一沉，心中升起了不好的感觉，脱口叫道：“公主姐姐怎么样了？你倒快说啊！”
唉！于一龙叹了一口气，皱眉道：“公主殿下她·······我们在撤出大营的路上，遇上了刺客······”“甚么？刺客！”
武天骄大惊，真是怕什么来什么，他最担心的就是刺客会趁着大营混乱的时候，伺机行刺。忙一催赤龙兽，贴近了于一龙，急切地问道：“公主姐姐现在怎么样？她有没有事？”
听到“公主姐姐”这么亲热的称呼，于一龙甚是不快，鼻孔中不由自主地哼出了一声，凛然道：“不劳金刀驸马爷挂心，公主殿下吉人天相，逢凶化吉，只是受了一点的轻伤，并无大碍，眼下正在山上的营帐里救治！”
闻听此言，武天骄松了一口气，悬着的一颗心放了下来，道：“如此幸好！如此幸好！不幸中的大幸！”
心念一动，问道：“于将军，刺客是甚么人？可有抓到刺客？”
于一龙摇了摇头，道：“本将军也不知道刺客是甚么人？行刺的刺客有十数人之多，个个都是超一流的杀手，我们死了不少的弟兄。幸好公主殿下身边有羿和和缪玉两位保护，才没让刺客得逞，不过，公主殿下还是被刺客的飞刀所伤，险些丧了命！”
啊！武天骄张大了嘴巴，心中骇然，暗道：“飞刀！真是玉姐吗？”
一念至此，一催赤龙兽，便往山上而去。于一龙见状忙拦住了他，沉声喝道：“你要干什么？”
武天骄一怔，忙道：“我去看看公主姐姐，看看她的伤势怎样？看我能不能帮上忙。”
“不用！你能帮什么忙，别添乱就行了！”
于一龙阴着脸，深沉地道：“公主殿下受了伤，除了公主殿下身边的人，任何人都不得靠近公主殿下。金刀驸马，你是男人，有道是男女授受不亲，男女有别，你最好不要去打扰公主殿下！”
真是狗嘴吐不出象牙，武天骄听这话怪刺耳的，浑身的不舒服，瞧见于一龙那阴沉的冷脸，心中暗骂：“什么狗屁东西，什么男女授受不亲，我看你是存心拦着不让我去见靖国公主！”
有心想闯上山去，但于一龙毕竟是靖国公主手下的大将，冒然得罪了他未免不好。
武天骄心中有气，上下打量了于一龙几眼，猛然间，心中泛起了一个疑问，寻思着：“这个于一龙到底是什么人？我住在樊家的日子里，他也住在樊家，樊夫人和靖国公主对他都很亲近，器重，他们之间似乎是亲戚关系，这个于一龙到底是樊家的什么人？”
这个疑问早在武天骄心中盘缠很久了，只是平日没有顾及，才没有深入研究，此时想起来不由得有点懊恼，心想：“我该问问乔影姐姐她们，也许她们知道一点！”
正想着，山上跑下了一匹马，马上是一位女将，正是天罡女卫的新任的暂代首领雷娜。雷娜一边跑，一边叫喊：“于将军！于将军······”声音中透着无比的急切、焦虑。
于一龙和武天骄见了都心中凛然，升起了不好的预感。于一龙忙拔马迎了上去，叫道：“雷娜姐，出什么事了？如此慌张？”
雷娜马到了于一龙近前停了下来，张了张嘴正想说，旋即意识到了什么，压低了声音轻声说着。她的声音甚低，但远处的武天骄聚精会神，运功倾听她说的话，听到：“于将军，情况不妙，公主殿下所中的飞刀淬有剧毒，公主殿下中毒了！”
啊！听到这话，于一龙和武天骄均感骇然，但闻于一龙急切地道：“飞刀有毒！公主殿下中的是什么毒？”
雷娜摇了摇头，蹙眉道：“我们也不知道是什么毒，这种毒十分奇怪，我们从未见过。所幸公主殿下内功深厚，已经运功暂时将毒性压住了，一时半会不会有什么生命危险，只是时间一久，公主殿下怕是压制不住，一旦毒发，公主殿下就有生命危险，我们要赶快给公主殿下请郎中解毒！”
“好！”
于一龙赶忙道：“我这就派人去附近的城镇把郎中找来！”
说着，叫来了四名皇鹰战士，对他们吩咐了几句。那四名皇鹰战士立刻飞身上马，疾驰而去。
武天骄见了直皱眉头，心想：“既然是奇毒，一般的郎中根本解不了，你们去请郎中，请再多的郎中来有个屁用！”
一想至此，催着赤龙兽往山上驰去。
